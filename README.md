# Moses Ipoba - Portfolio

This is a self-updating portfolio built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).  
It showcases my projects, experience, and skills as a Computer Science student and Full Stack Developer.

## Features

- **Dynamic Project Section:**  
  Fetches and displays your latest GitHub projects automatically.
- **Modern UI:**  
  Built with Tailwind CSS and custom components.
- **Dark Mode:**  
  Theme toggling with system preference support.
- **Responsive Design:**  
  Looks great on all devices.
- **Animated Hero Section:**  
  Engaging video background and interactive hero area.

## Tech Stack

- [Next.js 15](https://nextjs.org/)
- [React 19](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [GitHub API](https://docs.github.com/en/rest)
- [shadcn/ui](https://ui.shadcn.com/)
- [TypeScript](https://www.typescriptlang.org/)

## Getting Started

1. **Clone the repository:**

   ```sh
   git clone https://github.com/mosesipoba212/Self-updating-porfolio.git
   cd Self-updating-porfolio
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Set up environment variables:**

   - Copy `.env.example` to `.env.local` and fill in your GitHub API details:
   ```
   GITHUB_USERNAME=mosesipoba212
   GITHUB_TOKEN=your_github_token_here
   ```

4. **Run the development server:**
   ```sh
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the site.
